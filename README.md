# Unemployment Analysis in India During COVID-19

This project analyzes the impact of COVID-19 on unemployment trends in India using real-world data.

## Dataset
Source: Kaggle – Unemployment in India

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Key Insights
- Sharp rise in unemployment during 2020 lockdown
- State-wise differences observed
- Gradual recovery post COVID-19

## Repository Structure
- data/ – dataset
- notebook/ – analysis notebook
- report/ – final report and visuals
